# Workspace Memory

This repository is the workspace root for one or more project context banks, shared human-facing dashboards, and agent resources.

If a request could relate to more than one project, ask the user which project it belongs to before answering or making changes. Wrong-project edits are high risk.

Use `workspace_change_log/README.md` and `workspace_change_log/change_log.json` to track workspace-level changes that happen outside any individual project.
