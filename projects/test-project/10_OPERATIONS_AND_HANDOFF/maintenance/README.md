# Maintenance

Keep service procedures, inspection schedules, routine upkeep, and long-term care guidance here.

- Link to support notes for known problems and fixes.
- Keep one-time deployment or install steps out of this folder unless they are part of ongoing maintenance.

## Local Manifest

- Review `local_manifest.yaml` when entering this folder to quickly see whether any local files are relevant.
- Keep `local_manifest.yaml` up to date whenever files are added, removed, renamed, or materially repurposed.
