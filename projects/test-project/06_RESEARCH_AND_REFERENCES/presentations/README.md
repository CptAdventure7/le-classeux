# Presentations

Reference presentations and slide-based source material live here.

## Keep Here

- external slide decks
- presentation summaries
- talk notes used as supporting reference material

## Authoring Rules

- Store presentation-specific notes here instead of mixing them into `papers` or `web_references`.
- Link back to the original deck or event when available.

## Local Manifest

- Review `local_manifest.yaml` when entering this folder to quickly see whether any local files are relevant.
- Keep `local_manifest.yaml` up to date whenever files are added, removed, renamed, or materially repurposed.
