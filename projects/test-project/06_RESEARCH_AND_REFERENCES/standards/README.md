# Standards

Standards and compliance-reference notes live here.

## Keep Here

- standard summaries
- requirement-relevant extracts
- compliance interpretation notes
- standards comparison notes

## Authoring Rules

- Keep one file per standard or per tightly related standards topic.
- Link to the requirement, design, or verification work affected by the standard when relevant.

## Local Manifest

- Review `local_manifest.yaml` when entering this folder to quickly see whether any local files are relevant.
- Keep `local_manifest.yaml` up to date whenever files are added, removed, renamed, or materially repurposed.
