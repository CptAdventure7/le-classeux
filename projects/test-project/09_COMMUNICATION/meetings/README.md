# Meetings

Use this folder for meeting material only.

- Put transcripts, rough notes, and exports in `raw_notes/`.
- Put cleaned meeting outputs in `summaries/`.
- Keep raw notes and summaries linked when both exist.

## Local Manifest

- Review `local_manifest.yaml` when entering this folder to quickly see whether any local files are relevant.
- Keep `local_manifest.yaml` up to date whenever files are added, removed, renamed, or materially repurposed.
