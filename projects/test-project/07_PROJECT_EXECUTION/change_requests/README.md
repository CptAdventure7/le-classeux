# Change Requests

Keep formal change proposals here when the project needs to add, remove, defer, or re-scope work.

## Keep Here

- scope change requests
- schedule or budget change requests
- impact summaries linked to execution items or risks
- approval or rejection records

## Example

`2026-05-change-request-lab-integration-delay.md` describing the requested change, reason, impact, decision, and links to affected roadmap, execution-item, and risk artifacts.

## Local Manifest

- Review `local_manifest.yaml` when entering this folder to quickly see whether any local files are relevant.
- Keep `local_manifest.yaml` up to date whenever files are added, removed, renamed, or materially repurposed.
