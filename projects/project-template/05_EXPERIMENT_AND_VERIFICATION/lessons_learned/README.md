# Lessons Learned

Capture reusable takeaways here.

- Prefer one markdown file per lesson or theme.
- Update the existing file when adding more detail to the same lesson.
- Use a meaningful filename and link back to the related experiment, protocol, result, or analysis.

## Local Manifest

- Review `local_manifest.yaml` when entering this folder to quickly see whether any local files are relevant.
- Keep `local_manifest.yaml` up to date whenever files are added, removed, renamed, or materially repurposed.
