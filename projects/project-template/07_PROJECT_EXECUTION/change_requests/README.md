# Change Requests

Keep formal change proposals here when the project needs to add, remove, defer, or re-scope work.

## Keep Here

- scope change requests
- schedule or budget change requests
- impact summaries linked to issues, milestones, or risks
- approval or rejection records

## Example

`2026-05-change-request-lab-integration-delay.md` describing the requested change, reason, impact, decision, and links to affected roadmap, milestone, and risk artifacts.
