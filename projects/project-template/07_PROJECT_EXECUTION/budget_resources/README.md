# Budget Resources

Keep budget planning and resource allocation artifacts here.

## Keep Here

- budget snapshots
- staffing and capacity notes
- cost assumptions tied to roadmap or work packages
- resource tradeoff notes that affect delivery

## Example

`2026-04-budget-baseline.md` summarizing headcount, external spend, assumptions, and links to related roadmap or procurement artifacts.

## Local Manifest

- Review `local_manifest.yaml` when entering this folder to quickly see whether any local files are relevant.
- Keep `local_manifest.yaml` up to date whenever files are added, removed, renamed, or materially repurposed.
