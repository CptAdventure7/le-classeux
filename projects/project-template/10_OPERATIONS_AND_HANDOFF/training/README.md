# Training

Keep onboarding material, training plans, lesson outlines, and role-based training aids here.

- Link to user docs when training depends on day-to-day procedures.
- Keep operational troubleshooting in support notes, not in training material.

## Local Manifest

- Review `local_manifest.yaml` when entering this folder to quickly see whether any local files are relevant.
- Keep `local_manifest.yaml` up to date whenever files are added, removed, renamed, or materially repurposed.
