# Support Notes

Keep troubleshooting notes, known issues, workarounds, and support handoff information here.

- Link to release notes when a problem is introduced or fixed by a release.
- Move stable user-facing instructions into user docs once they stop being support-specific.

## Local Manifest

- Review `local_manifest.yaml` when entering this folder to quickly see whether any local files are relevant.
- Keep `local_manifest.yaml` up to date whenever files are added, removed, renamed, or materially repurposed.
