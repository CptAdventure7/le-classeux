# Competitor Landscape

Competitor research and market comparison notes live here.

## Keep Here

- competitor profiles
- feature or pricing comparisons
- market-position summaries
- notes that compare the project against named alternatives

## Authoring Rules

- Keep one file per competitor or comparison topic when possible.
- Link to supporting papers, standards, patents, or web sources instead of duplicating them here.

## Local Manifest

- Review `local_manifest.yaml` when entering this folder to quickly see whether any local files are relevant.
- Keep `local_manifest.yaml` up to date whenever files are added, removed, renamed, or materially repurposed.
