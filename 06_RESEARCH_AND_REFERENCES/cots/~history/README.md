# ~history

This folder belongs to the Maxwell context bank.

## What Belongs Here

Timestamped change notes for Cots. Each note records new information, its impact on existing information, the source type, the source link when available, and the capture time.

## Entry Format

- Create one new JSON file per meaningful update using `AAAA-MM-DD-HHhMM.json`.
- Store the entry as structured JSON, not prose markdown.
- Include `new_information`, `impact_on_existing_information`, `source_type`, `source_links`, and `time`.
- Use `source_links` as an array, even when there is only one link.

## What Does Not Belong Here

- Rewritten copies of the parent folder's main documents.
- Notes with no identified impact on the parent folder.
- Broad historical searches performed without strong evidence that this history is relevant.
